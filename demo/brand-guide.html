Brand guide

Visual identity for Yeti — the developer browser home station. Product name, logo usage, colors, and asset specs.

Mark: notch frame (left balanced) · v1 · June 2026

Logo mark

The primary mark is a shell frame ring with a left-edge notch — the same geometry as the in-product shell rim. Dark canvas shows through the center; the cream shape is the shell surface only.

Primary mark — 128px

Geometry is derived from frameShellRingSvgD scaled to a 32×32 viewBox. Icon container uses rx="8" (25% corner radius). Do not redraw the notch by hand — use the canonical path.

Clear space & minimum size

Keep a buffer of at least one rim thickness (7px in the 32px viewBox, ~22% of icon width) around the mark. No text, UI chrome, or other logos inside this zone.

Clear space (dashed)
  • Minimum icon size: 16×16px (favicon). Prefer 24px+ in UI.
  • Minimum lockup: 24px icon + wordmark at 14px/600.
  • Touch target: 40×40px minimum when the mark is tappable.

Colors

Brand colors are fixed for the logo. In-app theme presets (Work, Personal, Forest, etc.) change workspace chrome — the logo mark does not adapt to workspace themes.

Monochrome

Use primary everywhere possible. Monochrome only when color is unavailable (print, single-channel embeds). The mark-only variant (no container rect) is for animations and watermark overlays — never as favicon.

Typography

Yeti uses system UI fonts — no custom wordmark typeface.

Yeti

Role Spec Use
Wordmark system-ui / -apple-system, 600, −0.02em tracking Lockups, nav, footer
Product eyebrow 0.75rem, 600, 0.14em, uppercase Landing page header

Capitalise as Yeti in product copy. Lowercase yeti only for mechanical ids (package name, IndexedDB, snapshot filename).

Do & don't

✓ Shell ring with dark canvas center
✗ Solid blob with inward cut
✓ Use canonical SVG path
✗ Stretch, rotate, or skew
✓ Left-notch mark for brand
✗ Bottom-notch for brand mark (start page only)

Loading states

In-product loading is text-first today. When showing the mark, use a gentle opacity pulse — no spinners around the logo.
